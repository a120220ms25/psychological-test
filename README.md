# 💰 財富自由測驗

一個有趣的 H5 互動測驗遊戲，測測你離財富自由有多遠！

## 🌐 線上體驗

**👉 [立即體驗](https://a120220ms25.github.io/psychological-test/)**

## 📖 專案簡介

這是一個以財富和投資為主題的心理測驗遊戲，提供兩種不同的測驗類型：

1. **財富自由等級測驗** - 測試你目前的理財觀念和財富累積階段
2. **投資名人類型測驗** - 看看你的投資風格像哪位投資大師

## ✨ 功能特點

- 🎯 **雙測驗系統** - 兩種測驗各有 10 題，快速了解自己
- 📊 **5 個財富等級** - 從「吃土青年」到「財富自由預備軍」
- 👤 **6 位投資名人** - 巴菲特、索羅斯、馬斯克、彼得林奇、孫正義、雷達里奧
- 🎨 **簡約品牌色設計** - 清新專業的淺色系 UI
- 📱 **響應式設計** - 完美支援手機和桌面裝置
- 📸 **IG 分享功能** - 一鍵產生結果圖片並分享到限時動態
- 🔗 **連結複製** - 輕鬆分享測驗給朋友
- 😂 **幽默風趣** - 題目和結果描述充滿趣味

## 🛠️ 技術棧

- **HTML5** - 語義化結構
- **CSS3** - 簡約設計、動畫、響應式佈局
- **JavaScript (ES6+)** - 測驗邏輯、動態渲染
- **Inter Font** - 現代專業字體
- **PureCSS** - 輕量級響應式框架
- **html2canvas** - 結果截圖功能
- **Web Share API** - 原生分享功能

## 🎯 測驗系統

### 財富自由等級（5 個等級）

| 等級 | 分數範圍 | 說明 |
|------|---------|------|
| 😭 吃土青年 | 0-20 分 | 月底靠泡麵續命的理財新手 |
| 💸 月光貴族 | 21-40 分 | 會賺也會花的消費達人 |
| 💰 小資達人 | 41-60 分 | 開始有理財意識的儲蓄者 |
| 📈 理財高手 | 61-80 分 | 投資有道的財富累積者 |
| 🏝️ 財富自由預備軍 | 81-100 分 | 接近財富自由的投資高手 |

### 投資名人類型（6 種類型）

- 🤠 **巴菲特型** - 價值投資、長期持有
- 🦅 **索羅斯型** - 趨勢交易、順勢而為
- 🚀 **馬斯克型** - 顛覆創新、改變世界
- 🔍 **彼得林奇型** - 深入研究、生活選股
- 🎰 **孫正義型** - 大膽押注、高風險高報酬
- 🧭 **雷達里奧型** - 風險控制、資產配置

## 🎨 設計特色

### 色彩系統（V16 簡約品牌色風格）
```css
--bg-canvas: #DFDFDF;      /* 主體背景色（淺灰） */
--panel-color: #FFFFFF;    /* 面板/卡片背景色（純白） */
--text-dark: #333333;      /* 主要文字顏色（深灰） */
--accent-color: #D69396;   /* 品牌主色（柔和玫瑰粉） */
--shadow-color: #B07C7F;   /* 按鈕陰影色 */
--border-radius: 6px;      /* 簡潔圓角 */
```

### 互動效果
- **懸停 (Hover)** - 輕微抬升動畫，背景色略微變暗
- **點擊 (Active)** - 沉降效果，陰影消失
- **按鈕** - 專業的陰影沉降互動
- **卡片** - 簡潔的 hover 提升效果
- **字體** - Inter 現代專業字體

## 🔧 客製化

### 修改題目
編輯 `questions.js` 中的 `wealthLevelQuestions` 或 `investorTypeQuestions`：

```javascript
{
    question: "你的問題？",
    options: [
        { text: "選項 1", level: "等級名稱", score: 4 },
        { text: "選項 2", level: "等級名稱", score: 3 }
    ]
}
```

### 修改等級或類型
在 `questions.js` 中調整 `wealthLevels` 或 `investorTypes` 物件。

### 修改樣式
在 `style.css` 的 `:root` 區塊修改 CSS 變數。
